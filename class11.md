## Images

### controlling size of image in css:

```
img { 
width: 500px; 
height: 500px;} 
``` 
### aligning images using css:
``` 
img {
float: left;
margin-right: 10px;}
```
### Centering imagesUsing:

```
img { 
display: block; 
margin: 0px auto;} 
```
### Background Images:
```
body {
background-image: url("images/life.gif");}
```
