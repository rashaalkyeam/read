# HTML Element

## LIST
### There are many needs for HTMLL to contain lists , including three types:
1. Ordered lists
code: 
```
<ol>
<li>item</li>
<li>item</li>
<li>item</li>
</ol>
```
2. Unordered lists
code: 
```
<ul>
<li>item</li>
<li>item</li>
<li>item</li>
</ul>
```
3. Definition lists
```
<dl>
<dt>term</dt>
<dd>definition</dd>
<dt>term</dt>
<dd>definition</dd>
<dt>term</dt>
<dd>definition</dd>
</dl>
```
* Nested LIST: You can put a second list inside an ```<li> ``` element to create a sublist or nested list
